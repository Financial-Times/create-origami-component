# Usage
<!-- usage -->
```sh-session
$ npm install -g origami-authoring-tools
$ oat COMMAND
running command...
$ oat (-v|--version|version)
origami-authoring-tools/0.0.1 darwin-x64 node-v11.9.0
$ oat --help [COMMAND]
USAGE
  $ oat COMMAND
...
```
